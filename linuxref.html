Command Name Description Example
echo It repeats the message Echo hello world makes the computer say hello world in the next line.
date It tells us the hostname Entering hostname shows rpi08
arch print machine architecture entering arch shows armv7l
uname -a Print certain system information entering uname -a shows Linux rpi08 4.14.79-v7+#1159 SMP Sun Nov 4 17:50:20 GMT 2018 armv7l GNU/Linux
uptime uptime gives a one line display of the following current time, how long the system has been running, how many users are currently logged on, and the system load averages for the past 1, 5, and 15 minutes. Entering uptimes shows 16:17:57 up 13 days, 19:40, 1 user, load average: 0.25, 0.34, 0.37
who Print information about users who are currently logged in. Entering who shows sp1a pts/1 2019-04-29 15:26 (172.19.218.120)
id Print user and group Entering id shows uid=1002(sp1a) gid=1003(sp1a) groups=1003(sp1a),997(gpio),1007(studnetgrp)
ls Displays files in current directory ls websites
cd Moves user into chosen directory cd scripts moves the user into the scripts directory
